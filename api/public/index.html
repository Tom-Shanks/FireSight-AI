FireSight AI API

Welcome to the FireSight AI API. This API provides endpoints for wildfire prediction and monitoring.

Available Endpoints

GET /api/health

Check the health status of the API.

POST /api/predict/risk

Predict wildfire risk for a specific location.

GET /api/data/dashboard-stats

Get statistics for the dashboard.

GET /api/data/high-risk-areas

Get information about high-risk areas.

GET /api/data/recent-fires

Get data about recent fires.

For more information, see the GitHub repository.
